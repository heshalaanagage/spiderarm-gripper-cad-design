# Spider-Arm Gripper CAD Design

## Project Overview
This project contains the CAD design of a spider-arm style mechanical gripper developed using SolidWorks. The gripper mechanism was designed for robotic object handling and gripping applications with multiple articulated gripping arms.

## Design Purpose
The purpose of this design is to create a multi-arm gripping mechanism capable of securely handling objects with improved gripping stability and flexibility.

## Key Features
- Multi-arm spider-style gripping mechanism
- Mechanical linkage-based movement
- Compact gripper structure
- Symmetrical arm arrangement
- Designed for robotic and automation applications
- 3D modeled assembly with movable components

## Software Used
- SolidWorks

- SolidWorks part files

- Design screenshots / renders
